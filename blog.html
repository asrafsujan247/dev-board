Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM.
We can select an element with its tag name with getElementsByTagName() method. To select an element with its ID, you can use the getElementById() method. Also to select an element with class name use getElementsByClassName() method. There are also two methods, where we can select an element with their ID and class name. They are querySelector() and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The differences between innerHTML, innerText, and textContent are that with innerHTML we can get and set tags of an element and the text content as well. But in innerText we can't set or get the tag names of an element. With innerText we can only get or set the text content of an element. The textContent returns the text content also like innerText, but it also returns the text contents that are not rendered on the screen.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where we can set a event listener to its parent element to handle the events triggered on its child elements. Instead of adding event listeners to all the child elements we can delegate the event handling to their parent element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism that, if an event is triggered on a child element, it goes to the targeted element, and after checking the event it bubbles up or propagates upward to its parent elements to the root of the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create element using JavaScript with createElement() method. Also can add the created element by appending the child in the paren element with appendChild() method. And can also remove the element with remove() method in JavaScript.
